# Weather update

This is a weather app using:

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

Features:

- Current weather (temperature, humidity, description)
- 3-day forecast
- Dynamic animations for sun, moon, clouds, rain, snow

## How to view

1. Open `index.html` in a browser.
2. Or host live using [GitHub Pages](https://leya-dg.github.io/Weather/).# Weather
